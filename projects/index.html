Web Systems: Search Engine Project

As the final project of the course Web Systems, I was given the project specification of writing an end-to-end search engine. The search engine has several features including indexing implemented with MapReduce so it can scale to very large corpus sizes, information retrieval based on both tf-idf and PageRank scores, and a new search engine interface with two special features: user-driven scoring and summarization. The project was written using Python, React, Bash script, SQL, and Hadoop, and most of the codes were started from scratch. The website was then deployed to AWS for testing.
